# capsolver-npm-examples

### steps

1. do `npm install`
2. copy-paste `.env.example` to `.env`
3. add `APIKEY` and `PROXYSTRING` (proxy format `ip:port:user:pass`)

### usage

- run all sequentially: `./run-tests.sh`
